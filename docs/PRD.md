## 1. 背景与目标

这个项目缘起我个人的时间管理需要。v0.2 已经实现了日程删除、统计看板、CSV 导出、当日步骤汇总等核心功能。

v0.3 基于朋友反馈，做六项结构性改进：
1. 任务倒计时 — 条目上显示截止时间和剩余天数
2. 固定任务（周期性）— 循环任务一次设置、自动重复
3. 项目起始时间 — 项目级的可选起始日期
4. 单次任务拆分 — 今日/未来分区，支持添加任意日期的任务
5. 过期任务外观 — 逾期条目灰色 + ⚠️ 标记
6. 术语统一 — 单次任务、长期项目、固定任务、日历视图、数据统计

核心目标：从"当日记录"升级为"时间轴管理"。

## 3. 目标用户

大学生/职场新人，拖延倾向、记性不太好，需要一个简单工具可视化日常活动和想做的事。

核心场景：每天打开软件看一眼今天和之后要做什么；随手添加日程。

## 6. 功能详述

### 6.1 任务倒计时

每个单次任务和项目步骤条目上显示：
- 有截止日期：`⏰ 还剩 X 天` / `🔴 今天截止` / `⚠️ 已逾期 X 天`
- 无截止日期：不显示倒计时

### 6.2 固定任务

- **交互流程**：用户在"固定任务"标签页点击"添加固定任务"，弹出窗口设置任务名称、可选起止时间、可选时间时段、重复规则。起止时间可留空（表示不限），设置后日历视图中自动过滤起止范围外的日期，避免固定任务无限期显示。规则支持三种：每周（勾选星期几）、每个工作日（周一至周五，自动避开法定节假日）、每月（指定几号）。下方日历预览根据规则自动高亮对应日期，用户可点击日期自由增删。
- **列表显示**：条目右侧显示起止日期范围（如 `📅 2026-06-01～2026-12-31`），可留空。
- **边界情况**：工作日模式需依赖节假日数据，本期内置 2026 年中国法定节假日；日历预览范围为当前月开始 3 个月。

### 6.3 项目起止时间

添加项目时可填写可选的起始日期和截止日期（YYYY-MM-DD）。项目列表中起止时间显示在条目右侧（如 `2026-03-01 ~ 2026-06-30`），详情弹窗顶部显示并可编辑。

### 6.4 单次任务页拆分

页面分为上中下三区：
- 上区"今日任务"：当天任务 + 当天截止的项目步骤
- 中区"之后的任务"：未来日期的任务 + 未来截止的项目步骤
- 下区"过去的任务"：今天之前的任务（按日期降序），方便回顾过去做了/没做哪些事
- 添加今日任务使用简化弹窗；添加未来任务弹出带日历选择器的弹窗

### 6.5 过期任务外观

截止日期已过且未完成的任务/步骤：
- 文本变灰（#999999）
- 前缀加 ⚠️ 标记
- 长期项目：所有步骤都逾期且项目未完成 → 项目条目也标记过期

### 6.6 术语统一

详见第 2 节词汇表。代码中所有 UI 文本通过 `utils/i18n.py` 管理，为后续语言切换预留。

## 7. 验收标准

### 日常操作

- [x] **A-1 切换状态**：点击状态标记（✓/✗）立即切换，刷新后保持
- [x] **A-2 双击编辑**：双击弹出编辑窗口，预填内容（含截止日期）
- [x] **A-3 添加任务/步骤**：点击添加按钮，输入内容后出现在列表中

### 删除操作

- [x] **B-1~B-4**：删除任务/步骤/项目均有确认弹窗，未选中时提示

### DDL 汇总与同步

- [x] **C-1 当日截止步骤**：截止日期为当日的步骤出现在今日区域，标注所属项目
- [x] **C-2 项目自动完成**：步骤全完成 → 项目 ✓；步骤取消 → 项目 ✗

### 数据统计

- [x] **D-1~D-3**：折线图展示、范围切换、汇总信息

### 数据导出

- [x] **E-1~E-4**：导出弹窗、日期选择、CSV 内容、空数据提示

### v0.3 新增

- [x] **F-1 倒计时**：有截止日期的条目显示剩余天数/今天截止/已逾期
- [x] **F-2 固定任务添加**：能添加每周/工作日/每月规则的任务，日历预览正确高亮
- [x] **F-3 日历双向联动**：修改规则 → 日历刷新；点击日历日期 → 切换追加/排除状态
- [x] **F-4 固定任务日历集成**：主日历视图以紫色标记固定任务日期，点击日期在详情面板显示
- [x] **F-5 项目起止时间**：添加项目可设起始和截止日期，显示在条目右侧和详情页中
- [x] **F-6 单次任务拆分**：今日区域显示今日任务+步骤，未来区域显示之后的任务+步骤
- [x] **F-7 添加未来任务**：可指定日期添加单次任务，带日历选择器
- [x] **F-8 过期外观**：逾期任务灰色文字 + ⚠️ 前缀
- [x] **F-9 术语统一**：标签页名称和代码引用使用统一词汇
- [x] **F-10 固定任务起止时间**：添加/编辑固定任务可设起止日期，列表右侧显示日期范围，日历视图自动过滤范围外日期
- [x] **F-11 过去的任务**：单次任务页第三区显示今天之前的任务，按日期降序排列，支持切换完成/编辑/删除

### 非功能

- [x] **G-1 数据持久化**：关闭重开后数据不丢失，状态不变
- [x] **G-2 异常处理**：数据文件损坏时给出提示、以空数据启动
- [x] **G-3 向后兼容**：旧格式数据（无 routines 键）正常加载

## 8. 非功能需求

## 9. 风险与假设

- 当前支持数据范围为过去/未来 1 年
- 单用户本地使用
- 固定任务的工作日模式依赖手动维护的节假日数据（2026 年）
- 语言切换预留了 i18n 字典，翻译工作待后续版本
