# Changelog

本项目的所有重要变更记录。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [v0.3.1] — 2026-06-30

> 补丁版：两次 UX 打磨 + 数据架构稳定化。

### Changed
- 任务/步骤弹窗中"任务日期"与"截止日期"字段合并为"任务/截止日期"，统一使用日历选择器替代手动输入
- 数据目录从代码目录迁移至 `%APPDATA%\LightSchedule\`，开发模式与 exe 共享同一份数据，更新 exe 不影响用户数据

### Fixed
- 修复旧数据文件读取失败后未 return 导致 fall-through 到迁移逻辑的问题

### Removed
- 清除代码目录中遗留的旧数据文件（`schedule_data.json`、`schedule_data_backup.json`）

---

## [v0.3.0] — 2026-06-18

> 功能版：主线 6 项结构改进。从"当日记录"升级为"时间轴管理"。

### Added
- 任务倒计时：显示"还剩 X 天 / 今天截止 / 已逾期 X 天"
- 固定任务（Routines）：支持每周/工作日/每月/自定义规则，日历双向联动，可手动增删日期
- 固定任务起止时间：可选 `start_date` / `end_date`，超出范围的日期不在日历中显示
- 项目起止时间：项目增加 `start_date` / `end_date` 元数据字段
- 单次任务页分为三个区域：今日任务 / 之后的任务 / 过去的任务
- 过期任务外观：灰色文字 + ⚠️ 前缀标记

### Changed
- 标签页术语统一：每日任务 → 单次任务，新增固定任务标签页
- 窗口默认尺寸调整为 720×720

---

## [v0.2.0] — 2026-06-12

> 功能版：奠定架构基础，实现删除、统计、导出三项 Must 功能。

### Added
- 日程删除（任务/步骤/项目），含确认弹窗
- 统计看板：matplotlib 可视化，按类别 + 完成率展示
- CSV 导出（日程 + 项目汇总 + 步骤清单三个 sheet）
- 单元测试 15 个，覆盖 CRUD 全流程
- 日志系统（utils/logger.py）
- 项目自动完成：所有步骤完成时项目自动标记为完成
- 项目状态手动标记：有未完成步骤时弹窗确认
- PyInstaller 打包支持（`--windowed` 模式）
- PRD v0.2：7 条用户故事，MoSCoW 优先级排序

### Changed
- 架构重构：单文件拆为 6 个模块（`data_store.py` + `ui/*.py` + `utils/*.py`），数据层与 UI 层分离
- 步骤列表按截止日期排序
- 双击项目进入详情弹窗（替代按钮）

### Fixed
- 步骤排序后点击状态框操作错位（显示位置 → 数据位置索引映射）
- 步骤取消完成时项目状态同步变更
- 导出按钮叠放在标签栏右侧，避免单独占一整列

## [v0.1.0] — 2026-06-08

> 初始发布：最小可用产品。

### Added
- Tkinter 桌面应用，三个标签页：每日任务 / 长期项目 / 日历视图
- 每日任务：添加、勾选完成、删除
- 长期项目：添加项目、拆解步骤（含截止日期）、勾选完成
- 日历视图：tkcalendar 组件，任务日期与步骤截止日期以不同颜色标记
- 数据持久化：JSON 文件读写
