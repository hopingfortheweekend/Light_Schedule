# 轻简日程管理 (Light Schedule Manager)

一个基于 Python Tkinter 的本地日程管理工具。将大项目拆解为可执行的步骤，在日历上直观看到每个截止日期的时间分布，帮助拖延症患者一眼看清"我还有多少时间"。

## ✨ 功能

- **单次任务管理** — 添加任意日期的单次任务，支持截止日期和倒计时；今日/未来/过去三个区域分区显示，方便回顾
- **长期项目拆解** — 将大项目拆分为子步骤，每个步骤可设定独立的截止日期并显示倒计时；支持项目起止时间
- **固定任务（周期性）** — 支持每周/工作日/每月重复规则 + 日历预览双向联动；支持可选起止时间，到期自动停止
- **当日汇总** — 长期项目中截止日期为当日的步骤，自动显示在"单次任务"页面今日区域；固定任务同步出现在日历视图
- **倒计时 & 过期标记** — 每个任务/步骤条目显示距离截止还有几天；过期任务灰字 + ⚠️ 标记
- **日历可视化** — 所有日程按紧急度着色（逾期红→今天红→3天橙→7天黄→更远绿→固定任务紫），同一天以最紧急的为准
- **数据统计看板** — 折线图展示过去 N 周/月的完成 vs 全部趋势，含完成率概览
- **CSV 数据导出** — 按时间范围筛选并导出，包含单次任务、项目步骤和固定任务
- **本地数据存储** — JSON 文件持久化，数据完全由你掌控，无需联网

## 🛠 技术栈

| 层面 | 技术 |
|------|------|

## 🚀 快速开始

### 环境要求

- Python >= 3.8
- pip

### 安装 & 运行

```bash
# 1. 克隆项目
git clone <repo-url>
cd light_schedule

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python main.py
```

### 运行测试

```bash
pytest tests/ -v
```

## 📝 使用说明

| 标签页 | 操作方式 |
|--------|----------|
| 单次任务 | 上区今日、中区未来、下区过往；单击状态标记切换完成，双击编辑；添加今日/指定日期任务 |
| 长期项目 | 添加项目时可设起止时间；双击项目进入步骤管理，步骤按截止日期排序显示倒计时 |
| 固定任务 | 添加周期性任务时可设起止时间；选择每周/工作日/每月规则，日历预览自动高亮并支持点击切换日期 |
| 日历视图 | 按紧急度着色，点击日期查看彩色详情面板 |
| 数据统计 | 点时间范围按钮切换图表，查看完成率趋势（含固定任务执行次数） |
| 导出数据 | 点右上角"导出数据"，选范围 → 保存为 CSV |
